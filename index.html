Sensus

Sensus is an all-in-one event planning app designed to solve the problems of asynchronous communication and to leverage the power of data to smooth conflicts and inspire new event ideas. I am currently developing this application.

Imprivata Dashboard

I created a web dashboard to manage remote testing environments at Imprivata. On the front-end, it allowed users to reserve test machines, and examine stats on all the machines. On the back-end, a Mongo database held all this information, and updated it periodically. These updates were managed by Powershell and AutoIt scripts that automatically logged in and checked stats on all the machines in the remote environments.

I also created an Employee Shoutout dashboard with the same infrastructure to encourage Imprivata employees to thank colleagues publicly. I started all these pieces of the project with no prior knowledge of any of the tools I used.
